## Agent Types

### 1. Project Analysis Agent
Analyzes project health, timeline, budget, and risks.

### 2. Safety Compliance Agent
Monitors safety incidents, permits, and compliance.

### 3. Quality Control Agent
Automates inspection workflows and defect tracking.

### 4. Financial Agent
Tracks budgets, forecasts costs, manages cash flow.

### 5. Document Processing Agent
OCR, NLP extraction from construction documents.

### 6. Schedule Agent
Gantt optimization, critical path analysis.

### 7. Resource Agent
Optimizes labor, equipment, material allocation.

### 8. Communication Agent
Handles notifications, messages, stakeholder updates.

## Agent Configuration

Each agent has:
- `agent.json` - Agent metadata and configuration
- `system.md` - System prompt and behavior
- `tools/` - Available tools/actions
- `triggers/` - Auto-trigger conditions

## Subagents

Subagents are specialized agents for specific tasks:
- RFI Analyzer
- Change Order Processor
- Permit Tracker
- Incident Investigator
- Cost Code Analyzer
- Drawing Reviewer
- Weather Impact Analyzer
